部署环境

在GitHub上新建一个空的仓库 仓库名称为“用户名(自己的)”.github.io

配置git

git config –global user.name 你的英文名
git config –global user.email 你的邮箱
git config –global push.default matching
git config –global core.quotepath false
git config –global core.editor “vim”

创建新的空仓库，默认已经有本地仓库，上传本地仓库

git remote add origin git@github.com:xxxxxxxxx/仓库名.git
git push -u origin master

安装Hexo 依次食用

进入安全的目录，比如桌面
npm install -g hexo-cli 安装Hexo(global)
hexo init myBlog
cd myBlog
npm install
hexo new 第一篇博客 有一个md文件
start 第一篇博客.md 编辑md文件
start _config.yml 编辑网站配置
最后一行 type下 加上repo:(注意有一个空格)仓库地址(git@github.com:xxxxxxxxx)
npm install hexo-deployer-got –save 安装部署插件
hexo deploy
进入用户名.github.io

写第二篇博客

hexo new 第二篇博客
获取显示的路径 start 路径 编辑
hexo generate
hex deploy
第二篇成功
